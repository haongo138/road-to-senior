# Technical

Two tracks for senior SWE interview prep:

**Problem Solving** covers data structures, algorithms, and interview method. Start with the [DSA Roadmap](./problem-solving/dsa-roadmap) and the [Interview Process (UMPIRE)](./problem-solving/interview-process).

**Technical Concepts** is the knowledge round — concept Q&A on databases, caching, messaging, and distributed systems. Browse the [Technical Concepts hub](./concepts/concepts).
